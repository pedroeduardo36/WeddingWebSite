# **App Name**: Julia & Pedro Wedding

## Core Features:

- Countdown Timer: Display a countdown timer to the wedding date (May 2, 2026) on the home page.
- Our Story: Show the narrative 'Our Story' on the couple's page.
- Ceremony & Party Location: Embed a Google Maps to show the wedding location, along with important details.
- Dynamic Gift List: Presents (e.g., 'Honeymoon Fund') with progress bars, payment via PIX (QR code placeholder) and credit card.
- Progress Bar Logic: Simulate logic where collected value updates the progress bar using useState.
- RSVP Form: A form for guests to RSVP, including name, number of attendees, and dietary restrictions.

## Style Guidelines:

- Primary color: Off-white (#FAF9F6) for a clean, sophisticated background.
- Secondary color: Matte gold (#BDBDBD) for highlights and accents, evoking elegance.
- Accent color: Eucalyptus green (#77DD77) to bring a natural, refreshing touch.
- Headline font: 'Literata', a serif, for an elegant and classic touch to titles.
- Body font: 'PT Sans', a sans-serif, for readability and modern feel.
- Use Lucide-React icons throughout the site for a consistent and modern aesthetic.
- Generous use of whitespace to create a minimalist and elegant design.
- Subtle animations when interacting with the gift list or submitting the RSVP form.
